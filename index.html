What is HKOSCon?

Hong Kong Open Source Conference is an annual international open source conference in Hong Kong. It is aimed to demonstrate the use of open source software technology, open hardware technology and latest development in the world.

Themes

Cloud

Virtualisation Technology assists the growth of cloud computing and services. Open cloud computing provides a trust to users, deploying their business applications on cloud services. A wide ranges of open source projects are released in recent years from cloud infrastructure to devops services. At HKOSCon 2015, we would like to demonstrate the development of open cloud, and share knowledge of open source tools for cloud services.

Maker

Open source is the concept to open the design of the software products, and ask others for contribution to extend the creative opportunity of software product applications. Open Maker is the new trend to create creative ideas with open source concepts, using and developing hardware with open source software,and some of them are also open their hardware design. At HKOSCon 2015, we would like to demonstrate the unlimited possibilities of hardware and open source from makers.

Community: Source and Data

Open source development model driven the development of software in last 20 years, starting from free software commuity to re-write UNIX system tools, Linux kernel and distros, embedded systems, web and mobile applications, Android. Open source display the possibilities of crowd collaboration like bazaars by forming a number of developer and user communities, and open data is a recent trends to extend the concept of open source to data access in the world. At HKOSCon 2015, we would like to demonstrate how does open source model driven the development of applications, open data, communities, and society.

Startup

Many global and local startups uses open source software to develop their products and services for web and mobile users. Some of them created their public git repositories to release some open source projects. At HKOSCon 2015, we would like to demonstrate how does startups use and contribute open source software. And demonstrate the importance of open source software in the growth of startup businesses.

History of Conference

Idea of organising a open source conference in Hong Kong by community is started in 2004.

After a success in organising GNOME.Asia Summit 2012 in Hong Kong, open source community organised first community driven open source conference in Hong Kong in 2013.
