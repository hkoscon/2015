Openstack and containers

Speaker: Gustavo Panizzo (China)

Language: English (with English Slides)

Category: Cloud

Tag: meteorjs

Intermediate

About Speaker

i'm an openstack operator since Grizzly release (04/13), and linux guy since 2001. born in argentina, living in south china. i don't like caps.

About the Topic

I will talk about using containers (lxc, docker) to run openstack services, how to deploy them, how to upgrade, pros and cons.

Briefly I will talk about using containers to replace an hypervisor on openstack using technologies as lxd, lxc and libvirt.
