Keynote Speech: Data Transformation in Camel Style

Speaker: Roger Hui

Company and Title: Senior Software Maintenance Engineer, Red Hat Inc.

Language: English (With English Slides)

Tag: Apache Java JBoss

Beginners

About Speaker

Senior Software Maintenance Engineer with a focus on JBoss Fuse (based on Apache Camel and Karaf) and JBoss A-MQ (based on Apache ActiveMQ)

About the Topic

Data Transformation (converting POJO to XML or XML to JSON, etc) is tedious and boring but it is hard to avoid in the IT world, and this talk will show you how Apache Camel, Dozer, and the data-mapper tool make this mundane task easy and fun.
