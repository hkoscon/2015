Google Code-In — 開源，從小開始

Speaker: 海豹 & Denny Huang (Taiwan)

Community: SITCON

Language: Mandarin (With Chinese Slides)

Sub-Track: SITCON x HK

Tag: open source

Beginners

About Speaker

海豹（Jasper Yu, 尤理衡），目前就讀於高雄美國學校，ummm，應該算是國中生吧，以玩 Minecraft玩出一堆神奇的紅石指令機關在網路上發跡，後來不小心踏進了SITCON 的 Camp，然後就分不開了OAO ，之後又掉進了其他資訊與程式社群的無底坑。

Denny Huang，SITCON 發起人之一，SITCON 2013 與 2014 總召。

About the Topic

Google Code-In 是一個 Google 為了鼓勵全世界 13~17 歲學生參與 Open Source 所辦的比賽，透過與各大開源社群出題，給學生許多實務性的任務（包括程式、文件、資料…等）去完成。

議程將講述在 Google Code-In 中所完成的任務，並簡介其中幾個有趣的小專案，還有一般人不會做的遠距 pair programming 心得；透過 Google Code-In 認識實踐開放原始碼是件非常有趣的事，成為開源貢獻者就是這麼簡單！
